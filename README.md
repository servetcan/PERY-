🌿 PERY Skincare: Minimalist E-Commerce Web Concept
A premium, minimalist front-end web design concept for "PERY," a conceptual dermatological skincare brand . This project focuses on delivering a serene, nature-inspired user experience through clean typography, a muted color palette, and modern CSS techniques like Glassmorphism .

🚀 About the Project
PERY Skincare was designed with the philosophy that skincare—and its digital representation—should be calming, honest, and supportive . Built entirely with HTML5, CSS3, and Vanilla JavaScript, this 6-page conceptual website demonstrates strong capabilities in UI/UX translation, responsive layouts, and cross-page visual consistency .

✨ Key Features & Design System
🎨 Premium Aesthetic: Utilizes a carefully curated "Forest Green" (#2F5F4F) and "Light Beige" (#E0E0D8) color palette to evoke natural, dermatological trust .

🔤 Elegant Typography: Combines the sophisticated serif font Playfair Display for headings with the highly readable Open Sans for body text, creating a balanced, high-end editorial feel .

🪟 Glassmorphism UI: Implements advanced CSS backdrop-filtering to create frosted glass effects on the Contact and Product Detail pages, adding modern depth to the interface .

🔍 Search Simulation: Features a functional search input that captures user queries and simulates URL parameter passing (?search=...) to the products page .

📱 Responsive Layouts: Ensures a seamless browsing experience across all devices using flexible CSS Grid and Flexbox structures .

🔄 Dynamic Content Toggling: Uses Vanilla JavaScript to smoothly display specific product details (e.g., ingredients, usage instructions) without requiring a full page reload .

🛠️ Technologies Used
Frontend Structure: Semantic HTML5 

Styling: CSS3 (Custom Variables, Flexbox, Backdrop-filter) 

Interactivity: Vanilla JavaScript (ES6+)
